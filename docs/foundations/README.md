## Convention

Chaque fichier ici est **autonome** et **lisible séquentiellement**. Pas de résumé, pas d'extrait — du contenu structuré qui sert de référence stable.

**Snapshot historique** : à chaque cadrage stratégique majeur, créer une nouvelle entrée ou un nouveau fichier daté. Ne pas réécrire les fichiers antérieurs en gommant l'histoire.
