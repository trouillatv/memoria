## Convention des captures (à insérer)
Chaque chapitre contient des emplacements `📸` à remplir par une **capture annotée** :
1. **Capture** plein écran de la page, dans un **état avec des données** (jamais vide).
2. **Crop** sur la zone utile si besoin.
3. **Annotations numérotées** ① ② ③… pointant les boutons/zones clés (le texte de chaque numéro est fourni sous le bloc `📸`).
4. Déposer l'image dans `docs/guide/captures/` et la référencer dans le chapitre.

> Astuce capture (Windows) : **ShareX** ou **Greenshot** (gratuits) pour capturer + cropper + annoter en 2 min/écran. Le contenu, les libellés et les annotations sont déjà décrits dans chaque chapitre — il n'y a qu'à exécuter.
